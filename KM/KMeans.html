Clustering Introduction¶

Cluster Analysis is an Unsupervised Machine Learning Technique with no prior information given about the group or cluster members.Cluster is the collection of data objects which are similar to one another within the same group and are different between clusters. Cluster analysis involves formulating a problem, selecting a distance measure, selecting a clustering procedure, deciding the number of clusters, interpreting the profile clusters and finally, assessing the validity of clustering.

There are three primary methods used to perform cluster analysis:

  1. Hierarchical Cluster

Hierarchical procedure in cluster analysis is characterized by the development of a tree like structure. A hierarchical procedure can be agglomerative or divisive. Agglomerative methods in cluster analysis consist of linkage methods, variance methods, and centroid methods. Linkage methods in cluster analysis are comprised of single linkage, complete linkage, and average linkage

  1. K Means Cluster

The non-hierarchical methods in cluster analysis are frequently referred to as K means clustering.

  1. Two-Steps Cluster

Two-step cluster analysis is a combination of hierarchical and k-means cluster analysis

Project Overview¶

This project uses the fish data set as input to demonstrate how to perform cluster analysis. Even though we have label for this dataset but we will only be utilizing this information to validate our findings. We will visualize the clusters by using Linear discriminant analysis (LDA)

The Data:¶

Fish Data contains measurements of 159 fish caught in Finland's Lake Laengelmavesi.The following list describes each variable.

  1. species: species of fish
  2. weight : weight of the fish, in grams
  3. length1: length of the fish from the nose to the beginning of the tail, in centimeters
  4. length2: length of the fish from the nose to the notch of the tail, in centimeters
  5. length3:length of the fish from the nose to the end of the tail, in centimeters
  6. height: maximum height of the fish, in centimeters
  7. width: maximum width of the fish, in centimeters

Choosing a K Value¶

There is no easy answer of choosing a 'best' K value! we can plot within cluster SSE(sum of squared error) at different value of K and choose K at which the SSE decreases abruptly. This produces an elbow effect in graph and called elbow method.

In [27]:
from sklearn.cluster import KMeans
In [28]:
sse=[]
for i in range(1, 10):
    kmeans = KMeans(n_clusters=i, max_iter=1000).fit(df2)
    sse_i = kmeans.inertia_ # Inertia: Sum of distances of samples to their closest cluster center
    sse.append(sse_i)
In [29]:
plt.figure(figsize=(8,4))
plt.plot(range(1,10),sse,color='blue',linestyle='dashed',marker='o',markersize=8,
         markerfacecolor='red')
plt.title('SSE vs K Value')
plt.xlabel('K')
plt.ylabel('SSE')
plt.show()

We can see that optimal value of K should be around 4. We also know that in this particular data set, total number of species are 7. May be few species are very close to each other in term of features and can be represented by one Segment/Cluster.

Visualization of Clusters using Linear Discriminent Analysis (LDA)¶

Now we have labels for our data, however our data has multidimensional features and cluster cannot be viewed in two dimension.

To reduce dimensions, we can use Linear Discriminant Analysis. Linear Discriminant Analysis(LDA) is a very common technique used for supervised classification problems

The goal of Linear Discriminant Analysis is to project the features in higher dimension space onto a lower dimensional space

In [38]:
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
In [39]:
