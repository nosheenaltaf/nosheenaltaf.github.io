def text_process(raw):
    """
    1.case normalization
    2.remove integers
    3.remove punctuation
    4.remove stop words
    5.return list of clean text words

    """
    raw=raw.lower()
    raw = [i for i in raw if not any(j.isdigit() for j in i)]
    nopunc=[i for i in raw if i not in string.punctuation] 
    nopunc=''.join(nopunc) 
    nopunc=[i for i in nopunc.split() if i not in stopwords.words('english')] 
    text = [i for i in nopunc if len(i) >0] 
    text = [t for t in text if len(t) > 1]
    return text

TF-IDF Calculation¶

TF-IDF stands for term frequency-inverse document frequency

TF is calculated at document level

IDF Measure how common a term in entire corpus (weight of term in a corpus)

IDF is calculated at corpus level ( IDF close to 1 implies term is too common in corpus and IDF close to 5 means term is too rare in entire corpus)

The TF-IDF value is specific to a single document (d) whereas IDF depends on the entire corpus

def my_wordcloud(data):
    wordcloud = WordCloud(
        background_color = 'White',
        max_words = 200,
        max_font_size = 45, 
        scale = 3,
        random_state = 42
    ).generate(str(data))

    fig = plt.figure(1, figsize = (16, 10))
    plt.axis('off') 
    plt.imshow(wordcloud)
    plt.show()
